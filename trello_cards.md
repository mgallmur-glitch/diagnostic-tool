## 3️⃣ LISTA: 🏛️ METODOLOGÍA — Closing Cuántico

### TARJETA 13: Perfil 1 — Emocional Impulsivo
**Descripción:**
```
CARACTERÍSTICAS:
• Habla rápido (>150 palabras/min)
• Usa: sentir, emocionado, urgente, necesito, ya
• Quiere resultados AHORA
• Miedo: Perderse la oportunidad

ESTRATEGIA:
• Validar emoción primero
• Conectar pain con resultado emocional
• Cierre con compromiso emocional
• Ritmo: Rápido, energético
• NO dar tiempo de "pensarlo"

FRASE CLAVE:
"Siento esa urgencia, es señal de que estás listo"

SISTEMA DE CIERRE: E+PRD+ALE+PC (Estructura Gallmur)
```

---

### TARJETA 14: Perfil 2 — Racional Inseguro
**Descripción:**
```
CARACTERÍSTICAS:
• Habla moderado, pausado
• Usa: pensar, analizar, comparar, información
• Necesita datos, lógica, evidencia
• Miedo: Tomar decisión equivocada

ESTRATEGIA:
• Establecer lógica primero
• Presentar datos específicos (ROI)
• Reducir fricción con garantías
• Ritmo: Lento, pausado, dejar procesar
• Dar "permiso" para decidir

FRASE CLAVE:
"Te entiendo, quieres ver que los números funcionan"

SISTEMA DE CIERRE: Benjamin Franklin o Del 1 al 10
```

---

### TARJETA 15: Perfil 3 — Crítico Evasivo
**Descripción:**
```
CARACTERÍSTICAS:
• Habla lento con silencios
• Usa: depende, tal vez, consultar con
• Busca excusas para no decidir
• Miedo: Asumir responsabilidad

ESTRATEGIA:
• No aceptar excusas
• Firmeza sin agresión
• Reducir responsabilidad: "No decides ahora, solo comprometes a revisarlo"
• Ritmo: Firme, sin dejar escapar
• Forzar siguiente paso concreto

FRASE CLAVE:
"Entiendo que dependes de [persona], ¿y si preparáramos un resumen juntos?"

SISTEMA DE CIERRE: Del 1 al 10 + firmeza
```

---

### TARJETA 16: Perfil 4 — Decisor Dependiente
**Descripción:**
```
CARACTERÍSTICAS:
• Dice "nosotros" constantemente
• Menciona: mi esposa/socio debe verlo
• NO tiene poder de decisión real
• Otro persona decide

ESTRATEGIA:
• NO intentar cerrar en esta llamada
• Objetivo: Conseguir llamada conjunta
• "¿Podemos agendar 20 minutos donde [persona] también esté?"
• Preparar material para que "venda" al otro

FRASE CLAVE:
"Perfecto, ¿cuándo podemos hacer una llamada donde ambos estén?"

SISTEMA DE CIERRE: Ninguno (conseguir llamada conjunta)
```

---

### TARJETA 17: Los 6 Sistemas de Cierre
**Descripción:**
```
SISTEMA 1: "Si Yo Te Garantizo"
• Para: Racional Inseguro
• Estructura: Garantía específica o devolución

SISTEMA 2: "Del 1 al 10"
• Para: Prospecto interesado pero no compromete
• Estructura: Escala + "¿Qué te falta para el 10?"

SISTEMA 3: "Del Ladrón"
• Para: Compara con competencia o DIY
• Estructura: Honestidad brutal sobre tiempo/perdida

SISTEMA 4: "E+PRD+ALE+PC" (Gallmur)
• Para: Emocional Impulsivo
• Estructura: Emoción + Propuesta + Resultado + Decisión + Alegría + Paso + Cierre

SISTEMA 5: "Matrix"
• Para: Indeciso, "viendo opciones"
• Estructura: Comparativa forzada con criterios reales

SISTEMA 6: "Benjamin Franklin"
• Para: Racional que "necesita pensarlo"
• Estructura: Lista PROS vs CONTRAS
```
